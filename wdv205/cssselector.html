CSS Selector Specificity

Part One

Review and Research these selectors. Provide the following information for each of those selectors:

1) Describe what element or elements that selector will apply to a web page.

  • Element- one element based on name
  • Group- 2 or more elements, same style
  • Descendant- combines elements under a specified element
  • Class- selects elements to a specified attribute
  • ID- specified HTML attribute. One ID per page
  • Universal- selects all elements
  • Pseudo-links

    • 2) Discuss when to best use the selector.

  • Element- paragraphs or headers
  • Group- elements with the same color
  • Descendant- divs and paragraphs
  • Class- attributes that will go on multiple elements
  • ID- gradients
  • Universal- everything
  • Pseudo- changing colors of links

    • 3) What symbol or syntax identifies that selector on a CSS stylesheet?

  • Element-p{}
  • Group-h1, p{}
  • Descendent-div p{}
  • Class-.class
  • ID-#name
  • Universal-*
  • Pseudo-:
