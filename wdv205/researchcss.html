Research and Define CSS Terms

Specificity

If there are two or more rules in the same element, the browser chooses one. IDs are considered higher values and will win most the time. The universal sign is too broad and will lose

Precedence

How rules are cascaded in a website. It also states what rules are used in repetition. The order is: browser, external stylesheet, embedded styles, and inline styles

Inheritance

When a class is added to an element, the element becomes what the class says. For example, if you give a header a class that has a blue font color(.foo) and for a paragraph, you make a new class that says “inherit”, the paragraph with also have blue font.

Property

What you are changing. Properties are the elements like color or font. They are put before the values.

Value

What goes after the property. The property and value are separated by colons.

For example:

Color:blue;

Font-size: 12px;

Selector

To find and assign properties to element. This can be based on id, class, or attribute.
